\begin{abstract}
\noindent
This project investigates the problem of spam detection across multiple input types, including SMS, email, and YouTube comments, using machine learning models. The models leverage URL feature extraction and text vectorization techniques to provide an educated guess regarding the likelihood of spam. The results are analyzed to assess model performance and provide recommendations for improving spam detection accuracy.
\end{abstract}
